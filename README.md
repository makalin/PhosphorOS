# PhosphorOS Asset Pack

Retro terminal inspired starter assets for Titan 2 / Android launcher customization.

## Included
- 3 wallpapers
- 9 monochrome SVG icons
- KWGT starter text pack
- Nova Launcher starter config JSON

## Style
- Background: #000000
- Phosphor green: #00FF9C
- Amber: #FFC857
- Ice blue: #50DCFF

## Recommended setup
- Launcher: Nova Launcher
- Widgets: KWGT
- Font: VT323 or JetBrains Mono

## Notes
This is a starter pack, not a full Android theme APK.
