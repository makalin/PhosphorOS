## Design System

### Palette

- **Background**: `#000000`
- **Primary text**: `#00FF9C`
- **Accent**: `#FFC857`

### Typography

- **VT323**: retro terminal vibe
- **JetBrains Mono**: clean and highly readable

### Interaction style

- Text-first UI
- Keyboard-first navigation (Titan 2 friendly)
- Minimal chrome, no clutter
