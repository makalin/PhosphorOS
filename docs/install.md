## Theme Kit Installation (Nova + KWGT)

### Prereqs

- Nova Launcher
- KWGT (Kustom Widget)

### Steps

1. **Wallpaper**
   - Pick an image from `wallpapers/` (recommended: `phosphor_green.png` or `titan_grid.png`)

2. **Nova Launcher**
   - **Grid**: 4x5
   - **Labels**: ON
   - **Dock**: OFF
   - **Dark mode**: ON

3. **KWGT**
   - Use `kwgt/kwgt-starter.txt` as a starting point for your widget formulas.

4. **Icons**
   - Assign icons from `icons/` to match terminal labels (e.g. `web`, `mail`, `dir`, `snd`, `sys`).

### Notes

- The `launcher/` folder is a future Android launcher project and is not yet a buildable APK.
