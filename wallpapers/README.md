## Wallpapers

CRT / grid backgrounds for the PhosphorOS Theme Kit.

### Recommended

- `phosphor_green.png`: classic phosphor-on-black vibe
- `titan_grid.png`: minimal grid that keeps icons readable

### Tips

- If icons/text feel too bright, try reducing wallpaper brightness slightly in your gallery/editor.
- Pair with Nova dark mode and a mono font for the full effect.
