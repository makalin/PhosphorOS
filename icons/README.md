## Icons

Monochrome terminal-style SVG icons for the PhosphorOS Theme Kit.

### Usage

- Nova Launcher: long-press app → **Edit** → tap icon → **Gallery / Files** → choose an SVG-capable icon picker.
- If your launcher/icon picker can’t import SVG, convert to PNG (512×512) first.

### Design rules

- 256×256 viewBox with a black background
- Phosphor green stroke (`#00FF9C`), rounded caps/joins
- Stroke-only (no fills) so everything matches as a set
