## KWGT

This folder contains starter text formulas/snippets you can paste into KWGT.

### Files

- `kwgt-starter.txt`: original starter snippets
- `presets/`: organized snippets by widget type

### Import workflow (quick)

1. Create a widget in KWGT
2. Add a **Text** element
3. Tap the text → use the formula editor → paste snippets from `presets/`
4. Bind tap actions to apps as needed
